Car Service Record Management System

Validation Tests

Test Procedure

We'll schedule our validating testing efforts to ensure we're checking out the final, shippable product once development is finished. In order to guarantee that the system's many parts work together smoothly, we'll put them through their tests during Validation Testsing. Certain features will be tried that could not be tested before all required resources were available.

Certain non-functional criteria, such as performance, load bearing properties, and security, will also be tested. The testing methods that will be used for our validation checks are:
  • Functional Testing
  • Load Testing
  • Performance Testing

Test Cases

Functional Testing

We will validate the system by testing all of its functionality and checking that all of its requirements have been implemented as specified.

We will use load testing to make sure the system behaves as intended under stress and that it will gracefully shut down if it cannot handle the demand.
